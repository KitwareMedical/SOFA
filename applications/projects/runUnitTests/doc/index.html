runUnitTests

This project runs unit tests and can be used to check that function specs are satisfied.

Installation

  • Install boost::test    (package libboost-test1.46.1 on Ubuntu 12.04)
  • Uncomment  DEFINES += SOFA_PROJECT_runUnitTests in your sofa-local.prf file

Usage

The main program, located in the boost::test library, calls method init_unit_test( ) which takes a list of test libraries on the command line. A typical command line on Unix is:
bin/runUnitTest lib/*_test.so --log_level=message
This applies not only the standard Sofa tests, but also automatically performs all the plugin tests contributed by the plugin authors and compiled in your project.
A lot of command line arguments can be used to configure the behavior of the tests. Inside the tests, a variety of checkings with or without log messages can be issued.

Available tests

Library runUnitTests_SofaStandard_test aims at testing all the standard Sofa classes. Currently it tests only a few of them, and your contribution would be welcome.
Currently, the tests are:
  • Matrix_test : basic capabilites of th sparse matrices available in Sofa. Setting the matrix entries, matrix-vector products.
