Image MeshToImage_Frame Example


This example scene shows an example of the Image plugin, which computes a rasterization image from a mesh in simulation. You can indicate the moment (begin, end , step t) at which during the simulation the image is created. In this example, the output image is saved under the directory where you run Sofa. It is not shown in the image viewer. To run it, you need to load image Plugin and frame Plugin using the Plugin Manager of the GUI, or the command line.
More documentation can be found in ../doc/index.html
