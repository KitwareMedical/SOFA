Introduction

This plugin is documented in the following pdf: Flexible.pdf
To insert it in the SOFA development project, you can:
    1. add the following line to your sofa-local.prf
      DEFINES += SOFA_HAVE_PLUGIN_Flexible
    2. add the following line to Sofa.pro
      contains(DEFINES, SOFA_HAVE_PLUGIN_Flexible) { usePlugin(Flexible) }
    3. if your plugin depends on other plugins, edit file Flexible-dependencies.prf . See sofa-dependencies.prf for examples of how dependencies are defined.

Project structure

  • the C++ code is in the root directory
  • the documentation starts in doc/index.html and it is linked to the code documentation. The main page of this documentation is defined in file initFlexible.h.
  • a template of user documentation is found in Flexible.tex. Is should be filled in for every component in the plugin, then compiled with your favourite Latex distribution. The filename should match the name of the component.
  • the examples are in examples/ . The first one is Flexible.scn, and when runSOFA opens it, an html page pops up, with a link to the plugin documentation.
