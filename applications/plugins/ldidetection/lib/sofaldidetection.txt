INSTALLATION :
- Unzip the package in your $SOFA_DIR/applications/plugins directory. 
- You should have the following layout for your directory tree:
  -$SOFA_DIR/
    -applications/
      -plugins/
        -ldidetection/
        -plugins.pro
    +examples
    +...

- Add the following line to your plugins.pro
SUBDIRS += ldidetection
- edit ($SOFA_DIR)/sofa-default.cfg or ($SOFA_DIR)/sofa-local.cfg and uncomment 
the flag SOFA_HAVE_EIGEN2 (Line 92) it is not mandatory for compilation purposes, but the 
constraint response using LDI requires that flag.
- generate the makefiles as usual (on linux platform using sh $SOFA_DIR/Project\ Linux.sh)
- run the compilation 
- From the GUI in the edit/PluginManager dialog add the libsofaldidetection.so
- From the commandline runSofa -l ../lib/sofa-plugins/libsofaldidetection.so
- You are ready to use the examples provided in the ldidetection/examples directory.
